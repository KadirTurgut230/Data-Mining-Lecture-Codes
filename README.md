# Data-Mining-Lecture-Codes
This repository contains Python implementations of fundamental techniques in data mining and machine learning. Using the classic Iris dataset, the project provides a comprehensive examination of statistical analysis, various distance metrics, and both supervised and unsupervised learning algorithms, including k-NN, SVM, Random Forest, XGBoost, and K-Means.

Throughout the study, Stratified K-Fold Cross-Validation was employed to ensure robust model evaluation, and the performance of different algorithms was systematically compared under varying parameters. The codebase illustrates an end-to-end data science workflow, ranging from data preprocessing to comprehensive model assessment.

Veri Madenciliği Ders Kodları
Bu depo, veri madenciliği ve makine öğrenmesi süreçlerini kapsayan temel tekniklerin Python uygulamalarını içermektedir. Proje, klasik Iris veri seti üzerinde gerçekleştirilen istatistiksel analizler, çeşitli mesafe ölçüm metrikleri ve hem denetimli hem de denetimsiz öğrenme algoritmalarının (k-NN, SVM, Random Forest, XGBoost ve K-Means) kapsamlı bir incelemesini sunmaktadır.

Çalışma kapsamında, modellerin doğruluk oranlarını artırmak için Stratified K-Fold Çapraz Doğrulama yöntemi kullanılmış ve algoritmaların performansları farklı parametreler altında sistematik olarak karşılaştırılmıştır. Kodlar, veri hazırlama aşamasından model değerlendirme aşamasına kadar uçtan uca bir veri bilimi iş akışını örneklemektedir.
